Variable : is a memoryloaction where data is stored
	Type :	-> referrence types : address of value where it is
				-> class, interface, string etc..
			-> value types : values
				-> int, bool, long, byte, float, 
Data Type : it specify the type of the data your going store in a variable 
	
Operators : 
	-> Arthematic : + - % / * 
	-> Assignment : =, +=, -=, *=, /=, %=
			int age = 20;
			age += 4; age => 24
	-> Logical : && || !
	-> Unary : ++ --
	-> ternary : ?:
	-> Biwise : & | << >> ~ ^
	-> Relational : > < != == >= <=  

Conditional Statements : 
	-> if else
	-> switch
